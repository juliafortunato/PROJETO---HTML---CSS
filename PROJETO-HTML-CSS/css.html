O que é CSS?

Cascading Style Sheets (CSS), é um mecanismo para adicionar estilos/design a uma página web.

Propriedade e Valores

Propriedade: As propriedades CSS são os nomes que definem quais características de estilo serão aplicadas a um elemento HTML específico. Cada propriedade CSS tem um papel específico e pode aceitar diferentes tipos de valores.

Exemplo:

color: Define a cor do texto, font-size: Define o tamanho da fonte, background-color: Define a cor de fundo de um elemento.

Valores: Os valores CSS são os dados específicos fornecidos para as propriedades, definindo como essas propriedades devem ser interpretadas e aplicadas.

Exemplo:

Números, unidades de medida como pixels, cor, entre outros.

Classes e IDs

Classes: Classes são ideais quando você deseja aplicar estilos a vários elementos diferentes na página que têm características semelhantes.

IDs: IDs devem ser usados quando você precisa estilizar um único elemento específico na página que não se repete. Cada ID deve ser único em todo o documento HTML.
